# Hello World Extension

A simple Hello World VSCode extension.

## Features

- Registers a "Hello World" command that displays an information message

## Usage

1. Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on macOS) to open the Command Palette
2. Type "Hello World" and select the command
3. An information message will be displayed

## Development

```bash
npm install
npm run esbuild-watch
```

Then open the extension in VSCode by pressing F5 to start debugging.

## Building

```bash
npm run vscode:prepublish
```
