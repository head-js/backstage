# Tasks

## 执行模式

**必须遵守：**
- 严格按顺序执行 - 从第一个 `- [ ]` 开始
- 更新状态 - 完成后将 `- [ ]` 改为 `- [x]`
- 遇错即停 - 出错时立即停止，等待用户指示

**禁止行为：**
- 跳过任务
- 不按顺序执行
- 执行任务列表之外的工作
- 出错后继续执行
